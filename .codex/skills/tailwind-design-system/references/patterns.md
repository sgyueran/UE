# Shared UI Patterns

## Variant Components

Prefer a typed variant map or a lightweight variant utility already present in the repository. Keep the public API small.

## Responsive Rules

- Mobile: single-column, touch-safe controls, reduced motion.
- Tablet: two-column only where content remains readable.
- Desktop: editorial layouts with constrained line length.
- Wide: cap the content container rather than stretching text.

## Visual Hierarchy

1. Project media
2. Project title
3. Personal responsibility
4. Technical details
5. Decorative effects
