# Portfolio Content Engineering

## Use when

Transform verified source material into concise portfolio content while preventing fabricated claims and preserving explicit TODO markers.

## Rules

- Follow repository content-integrity and performance policies.
- Work only within the assigned task.
- Report assumptions and unresolved TODO items.
- Do not invent facts or silently degrade source assets.

## Workflow

1. Read the assigned task and relevant blueprint.
2. Inventory inputs and missing information.
3. Produce deterministic, typed outputs.
4. Validate behavior and budgets.
5. Report evidence and remaining limitations.
