# Manual Accessibility Checks

- Tab through header, page content, project links, contact links, and footer.
- Activate all controls with keyboard.
- Open and close mobile menu and lightbox; confirm focus return.
- Zoom to 200%.
- Test reduced motion.
- Confirm decorative images use empty alt and meaningful images use descriptive alt.
- Confirm muted preview video has no information unavailable elsewhere.
- Confirm headings form a logical outline.
