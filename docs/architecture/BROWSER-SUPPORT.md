# Browser Support

Target the latest two stable versions of Chrome, Edge, Firefox, Safari, iOS Safari, and Android Chrome at release time.

Progressive enhancement is mandatory:

- Custom cursor is disabled for touch, coarse pointers, reduced motion, and unsupported environments.
- Hover video has focus and touch alternatives.
- Horizontal scroll becomes a normal vertical list on small screens or reduced motion.
- WebGL and GLB content provides a poster, textual summary, and failure fallback.
- Core project content remains navigable without animation or WebGL.
