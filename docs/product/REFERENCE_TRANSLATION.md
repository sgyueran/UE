# JieJoe Reference Translation

## Purpose

Translate useful interaction principles from the reference experience into original UE5 portfolio rules.

Do not reproduce the original site's visual expression.

## Scene-like navigation

### Observed principle

Navigation can feel like entering a new scene or mode.

### UE5 translation

Treat the full-screen menu as a change in viewport context.

### Implementation direction

- full viewport layer;
- strong route hierarchy;
- current-route marker;
- project/contact shortcuts;
- ordered reveal;
- keyboard fallback;
- no literal Unreal loading-screen copy.

## Alternating interaction and calm

### Principle

Continuous stimulation reduces impact.

### UE5 translation

Alternate cinematic project media with quiet technical explanation.

### Rule

Do not place an animated gallery, parallax section, interactive diagram, and 3D viewer consecutively.

## Mobile feels authored

### Principle

Mobile should preserve identity while changing mechanics.

### UE5 translation

Use:

- touch-first scene transitions;
- large controls;
- simplified media;
- shorter narratives;
- fewer visual layers.

Do not merely shrink desktop layout.

## Interaction creates anticipation

### Principle

A subtle prompt before a meaningful response increases engagement.

### UE5 translation

Use cues such as:

- visible media edge;
- viewport marker;
- directional movement;
- concise “Explore system” action.

Do not hide essential content behind discovery.

## Transition continuity

### Principle

The user should understand that content changed without feeling abruptly teleported.

### UE5 translation

Keep shared anchors:

- grid fragments;
- route label;
- project media;
- navigation marker;
- camera or spatial relationship.

## Personal authorship

### Principle

A portfolio should not look assembled from popular component templates.

### UE5 translation

Build identity from:

- real project media;
- engineering diagrams;
- profiling captures;
- system explanations;
- personal technical language.

## What must not be copied

- page composition;
- exact transition choreography;
- illustrations;
- colors;
- typography;
- wording;
- navigation arrangement;
- custom assets;
- brand marks.

## Acceptance test

A reviewer should be able to say:

> The pacing feels intentional and immersive, but the visual language and evidence unmistakably belong to a UE5 engineer.
