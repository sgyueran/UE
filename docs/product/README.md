# Product Design Authority

This directory defines the product direction for the UE5 portfolio.

The reference website `jiejoe.com` is used as an interaction, pacing, and authorship reference. It is not a visual template to copy.

## Borrow

- immersive page rhythm
- scene-like navigation
- deliberate mobile interaction
- strong transition hierarchy
- moments of exploration followed by visual rest
- clear personal authorship

## Translate

- visual identity into Unreal Engine language
- motion into orientation, continuity, and feedback
- portfolio storytelling into engineering evidence
- mobile interaction into touch-first scene changes

## Do not copy

- exact layouts
- exact typography
- exact colors
- exact animations
- illustrations or artwork
- wording
- brand assets
- distinctive compositions

## Authority

For UI, content, navigation, and motion work, use these documents together with:

- `AGENTS.md`
- current Task Context
- page/component contracts
- verified content from `content-input/`

These documents do not override Allowed Files and do not authorize future-task implementation.
