# Codex Self-Review Prompt

Review your own implementation before finalizing.

Check:

1. Did you implement only the assigned Task?
2. Did you modify any unrelated file?
3. Did you introduce `any`?
4. Did you duplicate an existing abstraction?
5. Are keyboard and focus states correct?
6. Does mobile behavior work?
7. Is reduced-motion behavior present where needed?
8. Are event listeners and GSAP contexts cleaned up?
9. Did initial bundle size grow unnecessarily?
10. Do lint, typecheck, build, and tests pass?

Fix relevant issues before returning the final response.
