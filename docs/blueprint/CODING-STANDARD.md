# Coding Standard

- TypeScript only
- `strict: true`
- No `any`
- No duplicated component logic
- Keep page components below 300 lines
- Use named domain types
- Keep static content in `src/data`
- Use path aliases
- Handle loading, error, empty states
- Avoid direct DOM manipulation unless required by GSAP
- Clean up event listeners and animation contexts
- Do not modify unrelated files
