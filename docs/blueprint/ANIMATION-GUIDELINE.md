# Animation Guideline

## Libraries
- GSAP: complex timelines
- ScrollTrigger: scroll-linked sections
- Lenis: smooth scroll
- GSAP: timelines, transitions, and complex interaction motion

## Durations
- Fast: 0.2s
- Normal: 0.4s
- Slow: 0.8s
- Page transition: 0.7s

## Rules
- No page transition longer than 0.9s
- No forced horizontal scrolling on mobile
- Disable complex motion for reduced-motion users
- Always clean up GSAP contexts and ScrollTriggers
