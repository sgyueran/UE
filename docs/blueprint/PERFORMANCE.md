# Performance Standard

Targets:
- Performance >= 85
- Accessibility >= 90
- Best Practices >= 90
- SEO >= 90

Requirements:
- Route-level lazy loading
- Image lazy loading
- WebP/AVIF media
- Video `preload="metadata"`
- Three.js and GSAP split into separate chunks
- No GLB on initial homepage load
- Custom cursor uses requestAnimationFrame
- Avoid React state updates on every pointer frame
