# Design Language

## Palette
- Background: `#09090b`
- Surface: `#111114`
- Card: `#18181b`
- Border: `#27272a`
- Primary: `#3b82f6`
- Accent: `#a855f7`
- Text: `#f4f4f5`
- Muted: `#a1a1aa`

## Typography
- Display: Space Grotesk
- Body: Inter
- Technical Labels: JetBrains Mono

## Rules
- Large type, restrained neon
- Avoid excessive glow
- Preserve strong content hierarchy
- Use generous spacing
- Media is the visual focus
