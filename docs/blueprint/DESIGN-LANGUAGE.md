# Design Language

## Palette
- Background: `#09090b`
- Surface: `#111114`
- Card: `#18181b`
- Border: `#27272a`
- Primary: `#3b82f6`
- Accent: `#a855f7`
- Text: `#f4f4f5`
- Muted: `#a1a1aa`

## Typography
- Display: Space Grotesk
- Body: Inter
- Technical Labels: JetBrains Mono

## Rules
- Large type, restrained neon
- Avoid excessive glow
- Preserve strong content hierarchy
- Use generous spacing
- Media is the visual focus

## Reference Adaptation Rules

- Reference sites may inform pacing, orientation, navigation confidence, and mobile interaction patterns only.
- Do not copy external layouts, typography, color ratios, illustrations, wording, animation choreography, brand marks, or assets.
- The UE5 portfolio visual language must come from verified project media, viewport depth, system diagrams, Blueprint relationship expression, profiling evidence, and engineering constraints.
- Use near-black canvas, off-white text, restrained Unreal-inspired blue, and optional signal colors only when they communicate real state.
- Avoid generic cyberpunk neon, fake editor chrome, decorative HUD panels, and visual effects that replace project evidence.
- Mobile identity must be intentionally re-authored with fewer layers, clear touch targets, and no real horizontal overflow at 375 px.
